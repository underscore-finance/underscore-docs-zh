# 用户钱包：你的 DeFi 指挥中心

Underscore 用户钱包是一个智能合约钱包，通过单一地址统一访问顶级 DeFi 协议。在一笔交易中执行复杂的多协议策略。在严格边界内将操作委托给 AI 代理和可信操作员。资金在生息的同时自动化支付。在你的各个钱包间即时转移百万级资金。

基于 Base L2 构建，你的钱包结合了机构级安全性和 24/7 捕捉机会的灵活性——无论你是积极管理还是酣然入睡。

在其核心，Underscore 提供四个关键组件：

1. **统一协议访问**：通过标准化适配器直接集成顶级 DeFi 协议
2. **委托系统**：[管理员](managers.md)可在定义边界内执行策略
3. **支付通道**：自动化支付系统支持定期转账和一次性支付（详见[收款人](payees.md)和[数字支票](cheques.md)）
4. **安全层**：时间锁、白名单和细粒度权限保护每项操作

## 一个钱包，连接所有协议

Underscore 通过称为“Lego”的标准化适配器（乐高模块）连接到 DeFi 协议。每个 Lego 为协议特定操作提供一致的接口，实现原子化多协议交易并优化 Gas 使用。

**你的钱包未来无忧**：当新的 DeFi 协议发布或现有协议添加功能时，它们通过新的 Lego 集成自动可用于你的钱包。无需迁移资金、部署新钱包或更新任何内容——你现有的 Underscore 钱包在 Lego 部署的瞬间即可访问每个新协议。

### 收益与借贷协议

**集成协议**：Morpho、Moonwell、Aave、Euler、Fluid、Compound

**功能**：

- 存入资产赚取收益
- 自动解仓并提取资金
- 跟踪收益并计算利润
- 领取协议奖励
- 在协议间再平衡

**集成 DEX**：Aerodrome、Uniswap、Curve

**功能**：

- 通过智能路由兑换代币（最多 5 跳）
- 自动按比例添加/移除流动性
- 支持标准流动性（Uniswap v2）和集中流动性（Uniswap v3+）
- 管理 NFT 头寸并调整范围

### 债务管理

**集成协议**：Ripe Protocol

**功能**：

- 用抵押品借入 GREEN 或生息 sGREEN

- 赚取并领取 RIPE 奖励

### 资产转换

**功能**：

- ETH ↔ WETH 转换，零滑点
- 铸造和赎回凭证代币（如 stETH）
- 处理延迟赎回
- 自动格式转换以满足协议要求

### 奖励与激励

**功能**：

- 在一笔交易中批量领取所有协议的奖励
- 自动将奖励复利到生产性资产
- 跨协议跟踪终身收益

## 批量操作：多动作，单交易

Underscore 的架构允许将复杂的多步骤操作在单笔交易中原子化执行。这提供了显著的 Gas 节省并消除了部分执行的风险。

### 批量操作示例

**收益再平衡**：

```
1. 从 Aave 提取资金（较低收益）
2. 存入 Morpho（较高收益）
3. 从两个协议领取奖励
4. 将奖励转换为生产性资产
→ 一笔交易完成所有操作
```

**复杂建仓**：

```
1. 向 Ripe Protocol 存入抵押品
2. 以抵押品借款
3. 将借入资金提供为流动性
4. 质押 LP 代币获取额外收益
→ 一键完成策略
```

**投资组合再平衡**：

```
1. 从多个池中移除流动性
2. 将资产兑换至目标配置
3. 重新部署到新头寸
4. 领取并重新投资所有奖励
→ 整个再平衡原子化完成
```

## 其他钱包功能

### [管理员](managers.md)：委托操作

管理员是授权的操作员——人类或 AI——在你定义的边界内执行操作。他们可以交易、优化收益和管理支付，但不能提取到外部地址或超出你的限额。

**用例**：

- AI 代理 24/7 收益优化
- CFO 处理供应商和承包商支付
- 专业交易员管理部分投资组合
- 自动化债务头寸管理
- 家庭成员紧急访问权限

[→ 了解更多关于管理员](managers.md)

### [收款人](payees.md)：你的信任圈

收款人构成你可信的支付网络——这些地址只能接收你预先批准的款项。你的资金在支付时间前持续赚取收益，然后自动转账。只有你可以向这个圈子添加地址。

**用例**：

- 从生息资金支付员工工资
- 自动化供应商和承包商支付
- 具有拉取支付能力的订阅服务
- 带月度限额的家庭零用钱

[→ 了解更多关于收款人](payees.md)

### [数字支票](cheques.md)：可控的数字支票

数字支票将纸质支票的控制带入加密货币——开具收款人按其时间表兑现的支付，可在收款人兑现前随时取消。大额转账将自动触发安全延迟。你的资金持续赚取收益直到支付。

**用例**：

- 如果工作未交付可取消的承包商支付
- 带内置审核时间的大额转账
- 社交支付，如分摊午餐（给朋友 ¥200）
- 具有支付灵活性的一次性供应商发票
- 任何需要“撤销”选项的支付

[→ 了解更多关于数字支票](cheques.md)

### [白名单](whitelist.md)：无限信任

白名单是你的应急通道——白名单中的地址享有无限额、无延迟、无限制的转账特权。添加地址时采用时间锁定措施以防止被入侵，而你可以即时移除地址以保持控制权。

**用例**：

- 用于紧急资金访问的硬件钱包
- 需要立即整合的企业财务管理
- 用于风险分布的多钱包策略
- 如果你失去访问权限时用于恢复的 Gnosis Safe

[→ 了解更多关于白名单](whitelist.md)

## 常见问题

### 🔐 **安全与控制**

**这是自托管钱包吗？**  
是的，绝对是。你保持对私钥和资产的完全控制。Underscore 提供智能合约基础设施，但只有你可以授权交易。

**如果 Underscore 消失会发生什么？**  
你的资金仍然安全且可访问。智能合约是不可变的，不依赖于 Underscore 的服务器。你可以通过 BaseScan 或任何其他界面直接与你的钱包交互。

### 💰 **费用与收费**

**费用是什么？**

- **兑换费用**：代币兑换的小额百分比（可由协议配置）
- **奖励费用**：领取协议奖励时的小额百分比（可由协议配置）
- **收益费用**：对收益部分收取的小额百分比（可由协议配置）
- **收益分享**：高达 50% 的费用通过[奖励](rewards.md)返还给用户和大使
- **无费用项目**：转账、闲置资金、存款、债务操作、流动性提供、ETH/WETH 包装
- **完全透明**：每笔交易前显示确切的费用百分比

### 技术设置

**我可以使用现有钱包吗？**  
是的。你使用现有钱包（如 MetaMask）部署你的 Underscore 智能钱包，然后它作为所有者密钥。

**我可以访问哪些协议？**  
20+ 协议，包括 Aave、Morpho、Compound（借贷）、Ripe Protocol（借贷）、Uniswap、Curve、Aerodrome（交易）等。新协议通过 Lego 系统自动集成。

**如果 Base L2 有问题怎么办？**  
Base 继承以太坊的安全模型。在 L2 出现问题的极小概率情况下，存在将资产提取到以太坊主网的既定程序。你的资金始终在你的控制之下。
