# 奖励 (Rewards)：使用 DeFi 赚取收益

Underscore 与用户和大使 (Ambassador) 分享高达 50% 的协议收入。通过兑换手续费、实现盈利以及奖励领取来赚取收入——以你正在使用的同一资产形式支付。无需锁仓，无需归属计划，无需复杂的代币经济模型。真正的收益分享，源自真实的 DeFi 活动。

成为大使，当你的推荐用户使用协议时即可赚取收益。在盈利头寸上获得收益奖励。根据持仓累积存款积分。随时领取，直接转入任何白名单地址。你始终掌控资金，而产生的每笔费用都将成为赚取收益的机会。

## 收益分享机制

Underscore 对产生价值的活动收取费用，并通过收益分配器 (LootDistributor) 分配：

```
用户活动 → 协议费用 → 分配：
                   ├── 大使收益分享（默认 50%）
                   ├── 收益奖励（如已配置）
                   └── 存款奖励池
```

计算透明。分配自动进行。领取无需许可。

_注意：所有百分比均为默认值和示例。实际费率可由协议配置。_

## 大使收益：两条收入渠道

大使在推荐用户产生价值时获得收益：

### 1. 所有费用的收益分享

**自动获得每笔费用的百分比**（默认 50%，可配置）：

| 用户活动                | 协议费用    | 大使分成 (50%) | 示例                            |
| ----------------------- | ----------- | -------------- | ------------------------------- |
| $10,000（1 万美元）兑换 | 0.25% = $25 | $12.50         | 用户兑换代币                    |
| $1,000 盈利             | 10% = $100  | $50            | 用户实现盈利                    |
| $200 奖励领取           | 10% = $20   | $10            | 用户领取 MORPHO/WELL 等协议代币 |

### 2. 收益奖励（仅适用于盈利头寸）

**用户实现盈利时的额外奖励**（不适用于兑换或奖励领取）：

- 在盈利金额基础上额外获得 25% 奖励（典型示例，具体比例因资产而异）
- 可为生息资产、底层资产或替代代币（由协议配置）
- 要求：符合条件的收益头寸及可用分配器余额
- 当余额有限时，按先到先得原则分配

**示例**：用户实现 $1,000 盈利

- 用户保留 $1,000 盈利
- 大使获得 $250 奖励（盈利金额的 25%）
- 该奖励是额外的，并不影响 $50 的收益分享

**大使收入示例**：

假设每位用户每月为协议产生 $50 的费用：

- 协议收取：$50 费用
- 大使获得：$25（50% 收益分享）

**网络增长预测**：
| 网络用户数 | 月度收益分享 | 年度收益分享 |
|------------------|----------------------|---------------------|
| 5 名用户 | $125 | $1,500 |
| 10 名用户 | $250 | $3,000 |
| 20 名用户 | $500 | $6,000 |
| 50 名用户 | $1,250 | $15,000 |

_此外，当用户实现盈利时，还可获得潜在收益奖励（根据实际收益表现单独计算）_

## 用户收益：两种奖励机制

用户通过参与和表现获得收益：

### 1. 收益奖励（仅适用于盈利头寸）

**实现盈利时的直接奖励**（不适用于兑换或奖励）：

- 在盈利基础上额外获得 25%（典型示例，可配置）
- 在领取收益时自动发放
- 仅适用于收益策略，不适用于交易或奖励领取
- 要求：符合条件的收益头寸及可用分配器余额
- 示例：实现 $1,000 盈利 → 获得 $250 奖励

### 2. 存款积分系统

**基于持仓份额分享协议级奖励**：

```
积分 = (美元价值 × 持有区块数)
你的份额 = 你的积分 / 全网总积分
```

**运作方式**：

1. 在你的[用户钱包](user-wallet.md)中持有资产
2. 随时间累积积分
3. 协议将奖励添加至池中
4. 领取你应得的份额

_像支付宝集五福，但你的资金一直在赚钱，持有时间越长积分越多，但区别是你的资金始终在产生收益。_

**示例计算**：

- 持有 $10,000（1 万美元）持续 2 周 = 60.48 积分
- 全网总积分 = 3,024
- 你的份额 = 2%
- $5,000 奖励添加至池中 → 你领取 $100

## 费用结构

### 收取的费用

_所有费用均可配置。以下示例显示典型默认值。_

| 活动     | 费用范围      | 适用对象     | 示例                               |
| -------- | ------------- | ------------ | ---------------------------------- |
| 收益     | 10%（默认）   | 仅盈利       | $1,000 盈利 → $100 费用            |
| 兑换     | 0.25%（默认） | 兑换金额     | $10,000（1 万美元）兑换 → $25 费用 |
| 奖励领取 | 10%（默认）   | 外部奖励领取 | $200 奖励领取 → $20 费用           |

### 费用流向

所有费用通过透明分配流转：

- **大使分成**：50% 归推荐人（默认，可配置）
- **收益奖励**：若符合条件且可用
- **存款奖励池**：用于基于积分的分配
- **协议**：剩余部分用于运营

## 常见问题

**何时可以领取？**
费用生成后立即可领取，受冷却期限制。你可以授权[管理员 (Manager)](managers.md) 代为领取，但控制权始终在你手中。

**奖励会丢失吗？**
不会，奖励会累积至领取时（受合约余额限制）。

**积分如何运作？**
积分 = 存款价值 × 时间。存款越多 + 持有时间越长 = 积分越多。

**什么是冷却期？**
配置的冷却期，防止滥用同时允许定期领取。

**奖励是否总是发放？**
仅当分配器有足够余额时发放。先到先得。

## 核心要点

传统协议保留你产生的 100% 费用。Underscore 分享真实活动产生的真实收益。

**对于大使**：构建网络，永久获得所有费用的 50%（默认费率）。五个活跃用户 = $1,500/年。在此基础上扩展。

**对于用户**：你的存款赚取积分。你的收益赚取奖励。你的活动产生回报。

透明的费用分享，奖励参与。使用 DeFi，获得收益。推荐用户，获得更多收益。奖励以多种形式发放——你正在使用的资产、协议代币或其他有价值的资产。

这就是 DeFi 应该运作的方式——协议与用户共同成功。
