# Underscore Protocol：智能财务自动化

将你的静态钱包转变为动态财务操作系统。连接所有顶级 DeFi 协议，通过安全委托实现自动化策略，保持机构级安全标准——一切尽在单一界面。

**原子操作**：在单笔交易中跨五个协议再平衡 50 万美元。一键从 Aave 的 7% 年化收益率切换至 Morpho 的 12% 年化收益率。领取奖励、兑换为 USDC 并重新部署——原本需要 15 分钟和 5 次交易的操作，现在瞬间完成。

**安全委托**：让 AI 代理在凌晨 3 点帮你捕捉 15% 年化收益率的峰值，而你安心入睡。让你的 CFO（财务负责人）无需动用 200 万美元储备金即可支付 10 万美元供应商发票。设置精确仓位限制："交易 USDC 不超过 5 万美元，但绝不触碰我的 ETH。"每项操作都严格遵循你的规则，一键即可撤销。所有功能由 [Manager](managers.md) 驱动——你的全天候自动化操作员。

**统一协议访问**：告别标签页切换和钓鱼风险。你的 Underscore 钱包通过经过实战验证的适配器直接连接 Morpho、Aerodrome、Aave、Uniswap 及所有主要协议。在单笔交易中执行 Morpho 存款 → Aerodrome 兑换 → Aave 杠杆策略。

**智能支付通道**：使用[数字支票](cheques.md)向承包商支付，享有 72 小时的取消期限。将员工设置为[收款人](payees.md)，每月可精准提取 5,000 美元薪资（一分不多，一分不少）。通过[白名单](whitelist.md)在自有钱包间即时转移数百万资金。你的资金在被使用前持续赚取 5% 年化收益率。

钱包不该只是存钱的地方。它应该每时每刻都在为你赚钱——自动地。

## 核心组件

### [用户钱包：你的 DeFi 帝国大脑](user-wallet.md)

无需在安全与功能之间做选择——两者兼得。

- **单一地址**支持所有 DeFi 协议
- **智能自动化**，手动控制随时可用
- **机构级安全**，却无机构般的繁复
- **复杂操作**通过单笔交易完成

**适合：** 管理 1 万至 1000 万美元投资组合的活跃 DeFi 用户

### [管理员 (Manager)：按你的规则全天候运作的操作员](managers.md)

让人类或 AI 在你设定的严格边界内为你工作。

- **细粒度权限** - 只允许你指定的操作
- **硬性支出限额** - 每日、每月和累计限额
- **时间控制** - 设定期限后自动失效
- **即时撤销** - 你拥有绝对控制权

**适合：** 需要全天候执行交易的交易员，财务团队

### [收款人 (Payee)：自动支付，闲置资金持续生息](payees.md)

用可编程支付关系取代传统电汇。

- **预先批准的收款人**，每位都有独立限额
- **自动支付**，资金在等待期间持续赚取收益
- **支持拉取支付**，完美适配订阅服务
- **基于周期的限额**，防止超支

**适合：** 薪资发放、房租、订阅服务

### [数字支票 (Cheque)：立即发送，稍后确认](cheques.md)

带时间锁的支付承诺，在兑现前你始终掌控资金。

- **安全延迟**保护大额交易
- **支付前随时取消**
- **链上透明**，双方都能查看
- **资金不锁定**，直到实际支付

**适合：** 承包商、大额一次性支付、不太信任的收款人

### [白名单 (Whitelist)：可信地址的即时访问](whitelist.md)

为最信任的地址提供无限制访问权限。

- **无限制**转账给白名单地址
- **时间锁添加**（可配置）以增强安全性
- 需要时 **即时移除**

**适合：** 多钱包策略、硬件钱包、冷存储

### [奖励 (Rewards)：使用 DeFi 就能赚钱](rewards.md)

从协议活动中赚取真实收益。

- **50% 的收益分享**给予大使（默认）
- **收益奖励**基于实际利润
- **存款积分**，持有资产就能获得
- **无锁定期** - 随时领取

**适合：** 所有用户（通过日常活动获得被动收入）

## 谁在使用 Underscore

Underscore 不是消费级应用——它是驱动下一代金融应用的基础设施。把它想象成 DeFi 的 AWS——但完全非托管。

### 基于 Underscore 构建的应用

- **金融科技应用**：如 [Hightop](https://hightop.com)，利用 Underscore 实现收益优化、兑换、借贷及 AI 私人银行家功能
- **支付平台**：跨境汇款、B2B 支付及薪资系统，通过我们的[数字支票](cheques.md)和[收款人](payees.md)模块实现
- **智能钱包**：为现有钱包添加高级 DeFi 功能和[委托](managers.md)
- **AI 代理**：作为 [Manager](managers.md) 在用户定义的限制内运行的投资组合经理和交易机器人

### Hightop 示例

Hightop 展示了基于 Underscore 构建的强大功能：

- 他们的 **AI 私人银行家**是一个 Underscore Manager，全天候优化收益
- **用户提现**通过 Underscore 的白名单实现即时到账
- **所有 DeFi 操作**（收益、兑换、借贷）由 Underscore Lego 驱动
- 用户看到的只是一个简单的移动应用——Underscore 处理所有复杂性

大多数用户永远不会直接与 Underscore 交互。他们会使用像 Hightop 这样的应用，这些应用利用我们的基础设施提供无缝体验。

## 常见问题

**我的钱包可以升级吗？**  
不可以。出于安全考虑，钱包代码不可更改。新功能通过 Lego 集成实现。如果推出新功能，你可以迁移到新的 Underscore 钱包。

**什么是 Lego？**  
Lego（乐高）指的是与 DeFi 协议的标准化集成。新协议上线后，你的钱包立即可用。

**可以使用多个钱包吗？**  
当然可以。将它们加入白名单，即可在钱包间即时转账。

**支持哪些 L2 网络？**  
目前仅支持 Base L2。更多链即将到来。

**费用如何计算？**  
兑换收取小额费用（0.25%）、收益（10%）及外部奖励领取（10%）。转账、存款或闲置资金不收费。高达 50% 的费用通过收益分享返还给大使和用户。

**如果我丢失了所有者钱包怎么办？**  
如果你提前设置了 Manager 并添加了白名单地址，Manager 仍可将资金转到你的白名单地址（如硬件钱包），提供恢复路径。

**Manager 能偷走我的资金吗？**  
不能。Manager 的操作受区块链强制执行的限制。他们只能在你的支出限额内执行授权操作，无法提取到外部地址或修改自己的权限。

**如何处理紧急转账？**  
提前将你的硬件钱包或安全地址加入白名单。与有限额的收款人或有延迟的数字支票不同，白名单地址可以即时接收无限额转账，应对紧急情况。

## 支付方式对比

了解不同场景下该用哪种支付工具：

| 功能         | **[白名单](whitelist.md)** | **[收款人](payees.md)** | **[数字支票](cheques.md)** |
| ------------ | -------------------------- | ----------------------- | -------------------------- |
| **使用场景** | 即时无限转账至可信地址     | 自动定期支付            | 可控的一次性支付           |
| **支付限额** | 无——完全无限制             | 可按收款人自定义        | 每张支票设置限额           |
| **设置时间** | 时间锁添加（可配置）       | 立即生效（有限额）      | 无需设置                   |
| **核心优势** | 紧急访问及多钱包策略       | 设置一次，自动运行      | 兑现前随时取消             |
| **最适合**   | 硬件钱包、冷存储、其他钱包 | 薪资、房租、订阅        | 承包商、大额转账           |
| **风险等级** | 最高——可能清空钱包         | 中等——受限额保护        | 低——可取消                 |
